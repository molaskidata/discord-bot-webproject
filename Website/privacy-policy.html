Privacy Policy

Last updated: January 2025

1. Information We Collect

Coffee & Codes Bot collects minimal data:

  • Discord User IDs (for command functionality)
  • Server IDs (where the bot is used)
  • Message content (only for command processing)

2. How We Use Information

Data is used solely for:

  • Bot functionality and commands
  • Server moderation features
  • Improving user experience

3. Data Storage

We do not permanently store personal messages or user data. All data is processed in real-time.

4. Data Sharing

We do not share, sell, or distribute user data to third parties.

5. Data Security

We implement security measures to protect user data and follow Discord's guidelines.

6. User Rights

Users can request data deletion by removing the bot from their server or contacting us.

7. Contact

For privacy concerns, contact us on Discord: Coffee & Codes
